# Vintage-Daily-Operations-Dashboard
Designed and developed a daily operational dashboard for a customer service department, focused on email and phone support performance metrics.

Project Scope
Imported raw operational data from CSV files
Cleaned and transformed data using Excel Power Query
Calculated daily KPI and percentage-based performance metrics
Automated daily reporting workflow
Built interactive dashboards in Power BI
Designed a custom vintage newspaper-style user interface
Technologies Used
Microsoft Excel
Power Query
Power BI
CSV data sources
Data transformation
KPI analytics
Data visualization

Analytical Features
Daily KPI performance tracking

Dashboard Design
